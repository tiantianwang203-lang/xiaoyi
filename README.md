# 火星发展模拟器 — Mars Colony Simulator

> 2029年，第一批12名殖民者着陆火星乌托邦平原。500个火星日后，这里将拥有一座穹顶城。

## 快速开始

```bash
python mars_colony.py
```

从 **着陆→生存→发展→繁荣**，全程模拟火星殖民地建设。

## 关键里程碑

| 火星日 | 事件 |
|--------|------|
| Day 1 | 祝融号着陆乌托邦平原 |
| Day 5 | 首个温室建成，种下第一批土豆 |
| Day 50 | 首次火星土壤培育地球作物成功 |
| Day 100 | 第二批殖民者抵达（+8人） |
| Day 365 | 火星一周年，从生存迈向繁荣 |
| Day 400 | 穹顶城"新长安"封顶 |
| Day 500 | 殖民地完全自给自足 |

## 火星环境数据

- 重力：地球的 37.6%
- 平均气温：-63C（昼夜温差可达 160C）
- 一个火星日（Sol）：24.62 小时
- 沙尘暴季节：每 5-6 个地球月
